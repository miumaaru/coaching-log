ダブルループ学習

「どうすればうまくいくか」という行動レベルの改善（シングルループ）にとどまらず、「なぜそのやり方・目標・ルールを選んでいるのか」という前提や価値観ごと問い直す学習スタイル。繰り返す問題パターンの根本に触れ、本質的な自己変革をもたらします。
📚 教授の解説

理論の本質：なぜこれが必要なのか？

私たちは日々、仕事の中で「失敗→原因究明→改善→再実行」という流れで問題を解決しています。

たとえば、「書類ミスが多い→チェックリストを作る」というのが典型的な例です。これはシングルループ学習（single-loop learning：既存の枠内でやり方を改善する学習）と呼ばれます。速くて便利な方法です。

しかし、どれだけやり方を工夫しても、同じ問題が繰り返されることがあります。それは多くの場合、問題の原因が「やり方」ではなく、「前提（思い込み）」にあるからです。

ハーバード・ビジネス・スクールのクリス・アージリス（Chris Argyris）が1970年代に提唱したダブルループ学習（double-loop learning）は、まさにその「前提や価値観そのものを問い直す」学習です。

「なぜそのやり方を選んでいるのか」「そのルールはなぜあるのか」「自分は何を正しいと思い込んでいるのか」――こうした一段深い問いが、繰り返すパターンを断ち切り、根本的な自己変革につながります。

---

直感的な例え話：温度計 vs. 賢いサーモスタット

部屋が寒いとき、あなたならどうしますか？

シングルループは「温度計」です。「15℃だ→寒い→暖房をつける」。問題を検知し、決められた対応をする。正確で、速い。でも「設定温度が15℃で本当にいいのか？」は問いません。

ダブルループは「賢いサーモスタット」です。「なぜこの設定温度でいいのか？」「そもそも暖房より防寒着を増やす方が省エネでは？」と、設定値そのものを問い直します。

私たちの思考の中には、長年使い続けた「設定値」、つまり思い込みや価値観がたくさんあります。「これが当たり前」と信じて疑わないとき、私たちはシングルループのままです。

「そもそも、この考え方で合っているのか？」という問いが、ダブルループへの扉を開きます。

具体的なメカニズム

①
① シングルループ学習（行動の修正）
| 要素 | 内容 | |------|------| | 目的 | 既存のルール・目標の枠内で行動を改善する | | 問い | 「どうすればうまくいくか？（How）」 | | 例 | 「締め切りに遅れた→スケジュール管理アプリを使う」 | | 限界 | 同じ前提が続く限り、パターンが繰り返される |
②
② ダブルループ学習（前提の問い直し）
| 要素 | 内容 | |------|------| | 目的 | 行動を生み出している価値観・前提・ルールを問い直す | | 問い | 「なぜそれを目標にしているのか？（Why）」 | | 例 | 「なぜいつも締め切りに追われるのか→完璧にやろうとする前提があるのでは？」 | | 効果 | 根本的なパターンの変容が起きる |
③
③ 「防衛ルーティン」に気づく
アージリスは、自分の前提を問い直されそうになると、私たちが無意識に防衛ルーティン（defensive routines：正論ですり替える・話をそらす・「仕方ない」で終わらせる）を作動させると指摘しました。 「そんなことは分かっている」「この職場では仕方ない」「自分にはどうせ無理」――こうした言葉は、自分の前提を守ろうとする防御反応です。この反応に気づくことが、ダブルループの入り口になります。
🏋️ トレーナーの今日の課題

① ベビーステップ（2分以内）

このチャットを閉じたら、最近「また同じことになったな」と感じた出来事を1つだけ思い浮かべ、「このとき自分が当たり前だと思っていたことは何だろう？」と心の中で1回だけ問う（答えを出す必要はなし。問うだけでOK）

② ルーチン・トリガー

昼休憩にお茶・コーヒーを飲むとき、今週くり返している課題（小さくてもOK）を1つ思い浮かべ、「このパターンを生んでいる、自分の前提は何だろう？」と30秒だけ考える。考えが出てきたらメモする

③ 今日のミッション

今日の課題に取り組んでみましょう。

やり方を変えるだけでは足りない時がある。
「なぜそのやり方なのか」を問うことが、本当の変化を生む。

シングルループは「行動を修正する」。ダブルループは「前提を問い直す」。深い学習とは、自分の思考の根っこに触れることだ。
